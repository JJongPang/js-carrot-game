# 당근을 뽑자 게임

## 구현할 기능 목록
+ 게임 play 버튼 클릭시 bug, carrot 해당 필드에 랜덤 배치
+ 게임 Timer 구현
+ 게임 정지시 replay 가능
+ bug 클릭시 게임 종료(LOSE)
+ carrot 버튼 클릭시 게임 종료(WIN)
+ 상황별 팝업
+ 상황별 효과음 설정
